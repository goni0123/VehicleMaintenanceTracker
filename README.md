# VehicleMaintenanceTracker

## Description
A .NET Core Web API application to track vehicles, maintenance history, and crash logs.  
Built as a portfolio project to demonstrate ASP.NET Core, Entity Framework, and SQL skills.

## Features
- Add, update, and delete vehicles
- Record maintenance logs with date, type, cost, and mileage
- Record crash history with severity, cost, and repair details
- View vehicle history summary (total maintenance cost, accident history)
- Optional: user authentication for multiple users
- RESTful API endpoints for easy integration

## Tech Stack
- C# / .NET Core 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server (localdb)
- Git + GitHub for version control

## Setup Instructions
1. Clone the repository: `git clone https://github.com/yourusername/VehicleMaintenanceTracker.git`
2. Open the solution in Visual Studio 2022
3. Restore NuGet packages
4. Run EF Core migrations to create the database
5. Run the application
